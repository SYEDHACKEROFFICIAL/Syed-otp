## ✨ Features

- ✅ **Purple/Pink VIP Theme** - Premium dark theme with neon effects
- ✅ **User Registration System** - With admin approval workflow
- ✅ **Admin Panel** - Approve/reject users, manage numbers
- ✅ **Bot Configuration** - Per user Telegram bot setup
- ✅ **OTP Forwarding** - Auto forward OTPs to Telegram groups
- ✅ **Live OTP Stream** - Real-time OTP monitoring
- ✅ **Copy OTP Button** - One-click OTP copy with style
- ✅ **12+ API Integration** - Connected SMS APIs pre-configured
- ✅ **Per User Links** - Each user can set their own channel links
- ✅ **SMS History** - Store and view all received OTPs
- ✅ **Mobile Responsive** - Works on all devices
